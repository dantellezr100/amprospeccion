Seguridad por vendedor

El dispositivo se autoriza individualmente desde cada tarjeta de vendedor

Para activar/desactivar selfie, liberar dispositivo o hacer movimientos sensibles se solicita contraseña de super usuario. El primer celular usado por cada vendedor queda autorizado.

Borrar registros

Elimina todos los registros guardados en la nube. Requiere autorización de super usuario.

Acción irreversible: antes de borrar se descargará un respaldo, se pedirá contraseña de super usuario y confirmación escrita.
